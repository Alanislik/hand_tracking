# Hand Tracking / Gesture Control

Проект для управления курсором и действиями мыши с помощью жестов руки и/или головы.

## Возможности
- Управление курсором жестами
- Левый/правый клик, двойной клик, drag-and-drop
- Скролл жестами
- Режим управления головой (нос + моргания)
- Калибровка рабочей зоны в реальном времени

## Быстрый старт

### 1) Подготовка окружения
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

### 2) Запуск
```bash
python gesture2/main.py
```

При первом запуске автоматически скачаются модели MediaPipe в `gesture2/models/`.

## Полезные клавиши в приложении
- `H` — показать/скрыть help
- `C` — калибровка
- `R` — сброс калибровки
- `M` — переключение режима hand/head
- `A/D` — чувствительность
- `S/W` — сглаживание
- `P` — сохранить конфиг
- `Q` — выход
